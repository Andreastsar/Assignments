Default Positioning

Position: Default

Very Boring! The browser positioned the elements with the default way. That is from top to bottom. Not very funny!

Position: Absolute

Position: Absolute means that the browser places the different sections of the page at exactly the place you define using the top and left properties as in style2.css.

This of course, cannot solve all problems because it's very difficult to figure out the exact location of every element in a complicated web page.
