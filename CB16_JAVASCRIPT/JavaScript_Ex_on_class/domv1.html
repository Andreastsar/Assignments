Ex 9 - Document Object Model (DOM) 1

The DOM is your gateway to the contents of a web page and the key to making your pages come alive with JavaScript. The DOM is also an extremely detailed resource from which you can determine just about anything about the content and layout of a web page.

Window and Document Objects

The Window object is the main entry point to all client-side JavaScript features. This object is at the top of the scope chain and its methods and properties are essentially global variables and global functions.

The Document object is one of the most important properties of the Window object. It refers to Document object that represents the content displayed in the browser window.

The document.getElementById() Method

The easiest way to find an HTML element in the DOM, is by using the id element. If the element is found, the method returns the element as an object you can assign to a variable, otherwise element is null. We can use the getElementById() method of the Document object.

The document.getElementsByTagName() Method

If we want to find all HTML elements with the same tag we can use the getElementsByTagName() method.

The document.getElementsByClassName() Method

If we want to find all HTML elements with the same class name we can use getElementsByClassName() method.

The document.querySelectoreAll() Method

If we want to find all HTML elements that match a specified CSS selector (id, class, types) we can use the querySelectorAll() method.
