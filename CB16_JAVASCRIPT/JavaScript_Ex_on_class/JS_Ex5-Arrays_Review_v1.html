Level 0 - Ex 3 - More Array Methods

The Array.join() Method

The Array.join() converts all the elements of the array to strings and concatenates them, returning the resulting string. This method is the inverse of the String.split() method, which creates an array by breaking a string into pieces.

The Array.slice() Method

The Array.slice() method returns a sliced subarray of the specified array. Its two arguments specify the start and end of array slice to be returned.
